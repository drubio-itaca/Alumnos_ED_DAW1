# Portfolio Colaborativo - ED DAW 2025-2026

Este repositorio simula un entorno profesional de trabajo colaborativo utilizando Git y GitHub.

## Objetivo

Cada alumno deberá:

1. Hacer un **fork** del repositorio.
2. Crear una nueva rama con su nombre: tarjeta-nombreapellido
3. Añadir su archivo HTML dentro de la carpeta: alumnos/ (Ver ejemplo)
4. Añadir su tarjeta en el `index.html`.
5. Realizar commits claros y descriptivos.
6. Enviar un Pull Request para revisión.

---

## Reglas

Ten en cuenta que todos los alumnos de la clase van a estar trabajando en este repositorio. 
Sigue las siguientes normas para que funcione correctamente.
- No modificar tarjetas de otros compañeros.
- No modificar el CSS global.
- Añadir siempre tu tarjeta al final del bloque.
- Usar mensajes de commit claros.
- Trabajar siempre en una rama, nunca directamente en `main`.

## Resolver un conflicto

En algún momento el profesor realizará cambios en un archivo que también habrás modificado.
Cuando intentes actualizar tu repositorio, aparecerá un conflicto.

Deberás:
- Detectar el conflicto
- Editar el archivo manualmente
- Resolverlo correctamente
- Hacer commit del merge
